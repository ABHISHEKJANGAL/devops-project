# DevOps Project: Jenkins + Ansible

## Project Overview
Automated Apache installation and deployment using Jenkins pipeline and Ansible.

## How it Works
1. Developer pushes code to GitHub
2. Jenkins Pipeline triggers
3. Ansible playbook runs on remote web server
4. Apache installed, started, and index.html deployed

## File Structure
- apache.yml → Ansible playbook
- Jenkinsfile → Jenkins pipeline
- README.md → Project description
